Linux-toolbox provides easy and simple tools for Linux   

**Installation:**   

```sudo bash -c "$(wget -qO- https://raw.githubusercontent.com/Elpu7/linux-toolbox/main/install-toolbox.sh)"```   

**Execution:**   
   
```toolbox```   

**Update:**

```sudo toolbox update```

**Remove:**   
   
```sudo toolbox remove```
